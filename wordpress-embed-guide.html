WordPress Embed Guide: Crochet Pattern Challenge Game

Learn how to embed the interactive crochet game on your WordPress website

Method 1: Using Elementor HTML Widget

This is the easiest way to add the game to your WordPress site with Elementor.

Step 1: Copy the Embed Code

Copy the following HTML code:

<div class="crochet-game"></div>
<script src="https://REPLACE_WITH_YOUR_REPLIT_APP_URL/embedGame.js"></script>

Step 2: Add HTML Widget in Elementor

  1. Edit your page with Elementor
  2. Drag and drop an "HTML" widget onto your page
  3. Paste the embed code into the HTML widget
  4. Replace "REPLACE_WITH_YOUR_REPLIT_APP_URL" with your actual deployed Replit URL
  5. Click "Update" to save your page
Tip: Make sure the Elementor section width is set to at least 800px for the best game experience.

Method 2: Using a Shortcode (For Advanced Users)

If you want to use the game in multiple locations, you can create a shortcode.

Step 1: Add Code to Your Theme's functions.php

Add this code to your child theme's functions.php file:

function crochet_game_shortcode() {
  $output = '<div class="crochet-game"></div>';
  $output .= '<script src="https://REPLACE_WITH_YOUR_REPLIT_APP_URL/embedGame.js"></script>';
  return $output;
}
add_shortcode('crochet_game', 'crochet_game_shortcode');

Step 2: Use the Shortcode

You can now use the shortcode [crochet_game] anywhere in your WordPress site, including:

  • In regular WordPress posts and pages
  • In Elementor Text widget
  • In Elementor Shortcode widget
Important: Remember to replace "REPLACE_WITH_YOUR_REPLIT_APP_URL" with your actual deployed Replit URL.

Method 3: For Non-Elementor Pages

If you're using Astra theme without Elementor on some pages, you can still add the game.

Step 1: Edit the Page

  1. Edit the page using the WordPress block editor
  2. Add a "Custom HTML" block
  3. Paste the same embed code from Method 1
  4. Replace the URL with your actual deployed Replit URL
  5. Update the page

Customizing the Game Appearance

You can adjust how the game looks on your WordPress site by adding custom CSS in your theme.

Adding Custom CSS in Astra Theme

  1. Go to WordPress Dashboard → Appearance → Customize
  2. Go to Additional CSS
  3. Add your custom CSS

Example CSS for Customization:

.crochet-game-embed {
  /* Change max width */
  max-width: 800px;
}

.crochet-game-embed .game-header {
  /* Change header background color */
  background: #your-color-here;
}

.crochet-game-embed iframe {
  /* Change game height */
  height: 600px;
}

Troubleshooting

Game Not Loading

  • Make sure you've replaced the URL placeholder with your actual deployed Replit URL
  • Check if your Replit app is running and publicly accessible
  • Try clearing your browser cache

Game Appears Too Small

  • Adjust the container width in your Elementor settings
  • Customize the iframe height using custom CSS

Styling Conflicts

  • If your theme styling conflicts with the game, you may need to add more specific CSS rules
