Add an engaging crochet pattern quiz game to your website and increase visitor engagement.

== Description ==

The Crochet Pattern Challenge plugin adds an interactive and fun quiz game to your crochet or knitting website. This game tests your visitors' knowledge of crochet patterns, stitches, and techniques.

Visitors can select from three difficulty levels (beginner, intermediate, advanced) and answer multiple-choice questions about crochet. The game includes image-based questions, pattern recognition challenges, and stitch identification tasks.

### Features

* Multiple difficulty levels: beginner, intermediate, and advanced
* Beautiful visuals of crochet patterns and stitches
* Educational explanations and fun facts
* Score tracking and shareable results
* Fully responsive design for mobile and desktop
* Easy to embed with shortcode or Elementor
* Customizable height and theme

### How It Works

The game connects to a cloud-hosted game server where all the questions and scoring logic are managed. This means:

1. No impact on your WordPress site's performance
2. Regular updates to questions and features without plugin updates
3. Lightweight integration with your existing site

### Perfect For

* Crochet blogs and tutorial sites
* Yarn shops and craft stores
* Crochet pattern designers
* Knitting and craft communities

Increase visitor engagement and time spent on your site with this interactive crochet challenge!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/crochet-pattern-challenge` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Settings > Crochet Game to configure your game URL.
4. Add the game to any page using the shortcode `[crochet_game]` or the Elementor widget.

== Frequently Asked Questions ==

= Where does the game get its questions? =

The game pulls questions from a curated database of crochet patterns, stitches, and techniques. The questions are designed to be educational and engaging for all skill levels.

= Can I customize the appearance? =

Yes! You can choose different themes and set the height of the game using shortcode attributes:
`[crochet_game height="600" theme="blue"]`

= Will this slow down my site? =

No, the game is hosted on a separate server and loads asynchronously, so it won't affect your site's performance.

= Can I add my own questions? =

The current version uses a standard question set. Future updates may include the ability to add custom questions.
